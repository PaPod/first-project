Проект был нацелен на создание модели для предсказания вероятности снижения покупательской активности в течение трех месяцев на основе данных о поведении 1300 покупателей.

Основные этапы:  
- Подготовка данных: коррекция наименований категорий, замена выбросов в данных на медианы, устранение мультиколлинарности.  
- Моделирование: использовались модели логистической регрессии, дерева решений, kNN и метода опорных векторов. Для предобработки применяли MinMaxScaler и StandardScaler, кодирование признаков (OHE, OrdinalEncoder). Отбор признаков выполнялся через SelectKBest. Основная метрика — Recall, для минимизации пропуска покупателей с риском снижения активности.  
- Результат: лучшей моделью оказался kNN с параметрами: n_neighbors=2, weights='distance', k=9, Recall = 0.80.  
- Рекомендации: выявлены сегменты покупателей, находящихся под риском, с предложением персонализированных мер, включая рекомендации по продвижению альтернативных товаров и оптимизацию акций.
